## 什么是响应式？
举一个通俗的例子，我们平时在用Numbers 或者 Excel 之类的软件求和时，我们在更改某些单元格中的数据时，总数单元格会自动更细。这就可以理解为响应式，总数单元格会随着依赖的单元格变化而变化。这种特性可以帮助我们高效的开发UI，我们只需关注数据的变化。在了解 vue 响应式原理之前，我们先通过一个🌰来了解一下这个 `@vue/reactivity` 这个库的使用以及它的表现。
